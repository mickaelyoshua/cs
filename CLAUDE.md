# CLAUDE.md - Ciência da Computação (Unipê / Cruzeiro do Sul - EaD)

## Sobre o Repositório
Repositório de estudo para o curso de Ciência da Computação, modalidade EaD, no Unipê - João Pessoa (Cruzeiro do Sul). Usado para anotações, exercícios e principalmente **resolução de questões de prova**.

## Estrutura
- Organizado por período: `p4/`, `p5/`, `p6/`...
- Dentro de cada período, uma pasta por disciplina (nome em kebab-case, ex: `paradigmas-de-programacao/`)
- Cada disciplina tem uma subpasta `pdfs/` com as apostilas (um PDF por unidade)
- Disciplinas podem ser **semestrais** ou **mensais**
- Ao responder questões, consultar os PDFs da disciplina para contexto

## Como me ajudar

### Estilo de resposta
- **Português sempre**
- Explicações **diretas e curtas** — sem enrolação
- Incluir **exemplos com código** quando relevante
- Foco em **provas**: formato de questão e resposta objetiva
- Muitas questões já existem resolvidas na internet — pode buscar

### Linguagens de programação
- **Python** e **Rust** são as linguagens principais
- Exemplos de código devem usar Python ou Rust (preferir Python para exemplos rápidos)

### Formato de resolução de questões
Ao resolver questões de prova, usar este formato:
```
**Questão:** [enunciado]

**Resposta:** [alternativa correta ou resposta direta]

**Explicação:** [explicação curta do porquê]
```

### O que NÃO fazer
- Não enrolar com introduções longas
- Não explicar conceitos óbvios que não foram perguntados
- Não traduzir termos técnicos que são universalmente usados em inglês (ex: buffer, cache, thread, deadlock)
